# curriculAIk — Frontend

AI-powered teaching material generator. React frontend (EN/ES/CA).

## Setup

```bash
npm install
cp .env.example .env
npm run dev
```

## Structure
curriculaik-frontend/
├── src/
│   ├── components/    # Reusable UI components
│   ├── pages/         # Page components
│   ├── i18n/          # Translations (en, es, ca)
│   ├── services/      # API calls to backend
│   ├── hooks/         # Custom React hooks
│   └── assets/        # Images, fonts, icons
├── public/
└── .env.example

## Languages

The UI is available in English, Spanish and Catalan.
Translation files are in `src/i18n/`.
